# ColabCode

Run code server on Google Colab or Kaggle Notebooks

## Getting Started
Install colabcode 
```bash
pip install colabcode
```
Import colabcode
```python
from colabcode import ColabCode
```
Run colabcode
```
ColabCode(port=10000, password="myawesome password")
```
You can also run it with any password or port :)
